### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（雇员）、雇主、管理员。


**雇员功能：**
1. 登录
2. 注册
3. 首页
4. 服务项目
5. 需求信息
6. 网站公告
7. 意见反馈
8. 个人中心
9. 我的收藏




**雇主功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 需求信息管理
6. 服务预约管理
7. 申请预约管理
8. 签订合同管理
9. 电子签名管理
10. 雇主评价管理
11. 咨询雇员管理
12. 咨询雇主管理


**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息)
3. 雇主管理
4. 雇员管理
5. 资料认证管理
6. 项目类型管理
7. 服务项目管理
8. 需求信息管理
9. 服务预约管理
10. 申请预约管理
11. 签订合同管理
12. 电子签名管理
13. 雇主评价管理
14. 意见反馈
15. 系统管理（轮播图管理、网站公告）
